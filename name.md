Lattice — 晶格。每个 Agent 是晶格中的一个节点，ACP 协议是连接键，平台是整个晶体结构。从 AI Agent 编排自然延伸到半导体物理（硅晶格）、GPU 并行计算（网格架构），叙事空间最大。5 字母，科学感强，非军事。
Ingot — 硅锭。所有芯片的源头原材料（硅锭 → 切片 → 晶圆 → 芯片），平台 = Agent 的源头环境。5 字母，短小有力，工业感强，没有任何科技产品使用这个名字。
Zircon — 锆石。晶体结构矿物，ZrO₂ 用于芯片封装基板。和 Lattice 同属晶体学语系，晶格 + 晶体的品牌体系。6 字母，辨识度高，科技圈几乎没人用。
Cobalt — 钴。半导体材料（Co 用于芯片互连线路），钴蓝 = 高端品牌配色。元素名自带科技感，6 字母，读音响亮，视觉上适合做深蓝色调 logo。科技圈没有同名 AI/Agent 产品。
Anneal — 退火。唯一一个同时串起三个世界的词：芯片退火工艺（激活掺杂剂、修复晶格损伤）、冶金热处理（消除内应力）、AI 优化（模拟退火算法）。6 字母，动词感强，没有任何科技公司用过。
