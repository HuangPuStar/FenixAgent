# 智能体模板

智能体模板是 Agent 的配置模板，定义了模型、提示词、知识库和权限等设置。新建会话时可以基于某个模板创建 Agent 实例。

## 智能体列表

点击左侧导航栏的「智能体」进入管理页面。列表显示以下信息：

| 列 | 说明 |
|---|---|
| 名称 | Agent 唯一标识，创建后不可修改 |
| 类型 | 内置（系统预置）或 自定义（用户创建） |
| 模式 | primary / subagent / all |
| 知识库 | 绑定的知识库数量 |
| 默认 | 标记 ★ 表示为默认模板 |
| 模型 | 使用的 AI 模型 |

列表支持搜索和批量选择。

## 新建智能体

1. 点击右上角「新建智能体模板」
2. 填写基础配置、绑定知识库、设置权限
3. 保存

**名称规则**：只能包含小写字母、数字和单连字符（如 `my-agent`），长度 1-64 字符。

## 编辑智能体

点击列表右侧的「编辑」按钮修改配置。

编辑对话框包含三个标签页：

### 基础配置

| 字段 | 说明 |
|---|---|
| 名称 | Agent 唯一标识（编辑时不可改） |
| 模型 | 使用的 AI 模型，需先在模型管理中配置 |
| 模式 | primary（主 Agent）/ subagent（子 Agent）/ all |
| 步数 | 最大思考步骤数（1-200），简单任务 10-30，复杂任务 50-100 |
| 提示词 | 自定义 Agent 提示词 |
| 描述 | Agent 的简短描述 |
| Variant | 变体标识，如 `thinking` |
| 温度 | 生成随机性（0-2），越高越随机 |
| Top P | 核采样参数（0-1） |
| 颜色 | 界面显示颜色，hex 值或预设色名 |
| 隐藏 | 不在列表中显示，但仍可通过 API 调用 |
| 禁用 | 完全不可用，界面和 API 都无法使用 |

### 知识库

选择要绑定的知识库，并配置检索策略：

- **优先检索知识库**：开启后先从知识库检索，再进行 Agent 推理
- **最大返回条数**：每次检索最多返回的条数（1-20）

### 权限配置

控制 Agent 可以执行哪些操作。权限分为三级：

| 级别 | 行为 |
|---|---|
| 允许 | 直接执行，不询问 |
| 询问 | 每次执行前弹窗询问用户 |
| 拒绝 | 禁止执行 |

**全局策略**：设置后所有工具继承此策略。未设置则使用 Agent 引擎内置默认值。

**工具类型**：

- **开关型工具**：整工具只能设置一种权限
  - todowrite、question、webfetch、websearch、codesearch、doom_loop

- **规则型工具**：支持通配符规则，可以针对不同路径设置不同权限
  - read、edit、glob、grep、list、bash、task、external_directory、lsp

规则示例：允许读取所有文件，但拒绝读取 `.env` 文件：

```
read: { "*": "allow", "*.env": "deny" }
```

**Skill 权限**：控制 Agent 调用哪些 Skill。同样支持精确名称匹配和通配符规则。

## 设为默认

某个智能体被设为默认后，新建会话时会优先使用该模板。点击列表中的「设为默认」按钮即可。

## 删除智能体

自定义的智能体可以删除。内置智能体（如 `build`、`plan`、`general`）不能删除，只能修改配置。

- 单个删除：点击列表右侧的「删除」按钮
- 批量删除：勾选多个智能体后，点击底部批量操作栏的「批量删除」
