# 智能体编排

智能体编排功能通过 **iframe 嵌入**外部 acpx-g 服务实现。FenixAgent 控制面板只负责加载和显示编排界面，所有编排操作（节点设计、连线、运行等）都在 acpx-g 服务中完成。

## 访问方式

登录 FenixAgent 控制面板后，点击左侧导航栏的 **智能体编排** 进入编排页面。

## 界面状态

FenixAgent 端会显示以下三种状态：

| 状态 | 说明 |
|------|------|
| 加载中 | 显示加载动画，等待 acpx-g 服务响应 |
| 连接失败 | 显示错误提示和重试按钮 |
| 正常 | 显示 acpx-g 的编排界面 |

## 连接失败处理

如果页面显示"连接失败"，可能原因：

1. acpx-g 服务未启动
2. acpx-g 服务地址配置错误
3. 网络连接问题

**解决方法**：

1. 检查 acpx-g 服务是否运行（默认端口 8848）
2. 确认 FenixAgent 配置中的 `ACPX_G_URL` 环境变量正确
3. 点击 **重试** 按钮重新连接
4. 查看 FenixAgent 和 acpx-g 的日志获取详细错误信息

## 编排功能说明

具体的编排操作（节点设计、连线配置、工作流运行等）请参考 **acpx-g 服务文档**。FenixAgent 仅提供容器和访问入口，不参与编排逻辑的实现。
