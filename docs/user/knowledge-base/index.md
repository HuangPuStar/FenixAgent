# 知识库

知识库用来存放各类文档和代码资源，Agent 可以检索这些内容来回答问题。

## 页面布局

页面左侧是知识库列表，右侧是详情面板。

列表每行显示：

| 字段 | 说明 |
|------|------|
| 名称 | 知识库名称 |
| 状态 | 当前状态 |
| 资源数 | 已上传的资源数量 |
| 最近更新时间 | 最后更新的时间 |
| 绑定Agent数 | 使用此知识库的 Agent 数量 |

列表上方有搜索框和排序、筛选功能。

## 创建知识库

点击右上角 **新建知识库**，填写表单：

| 字段 | 说明 | 示例 |
|------|------|------|
| 名称 | 显示名称 | `项目文档` |
| Slug | 唯一标识符 | `project-docs` |
| 描述 | 可选说明 | `项目开发相关文档` |

Slug 只能包含小写字母、数字和连字符。

## 上传文件

两个入口可以上传文件：

1. 列表行的 **上传** 按钮
2. 详情页的 **选择文件** 按钮

支持多选，上传时会显示进度。

支持格式：PDF、Markdown、TXT 以及各类源代码文件。

## 导入 URL

在详情页输入 URL，可选填来源名称，点击 **导入 URL**。

URL 导入是异步处理，后台完成。

## 管理资源

详情页的资源列表显示所有已上传的资源，包含：

- 来源名称（文件名或 URL 来源名称）
- 状态（成功/失败/处理中）
- 来源类型（file/url）
- 更新时间

资源列表有滚动区域，最多显示 320px 高度。

### 删除资源

点击资源右侧的 **删除** 按钮，确认后删除。

### 资源状态

| 状态 | 说明 |
|------|------|
| 成功 | 资源已成功索引，可供 Agent 使用 |
| 处理中 | 资源正在后台处理 |
| 失败 | 索引失败 |

### 处理错误

如果资源显示失败状态：

1. 查看资源卡片中的错误信息
2. 根据错误提示修正问题（如文件格式不支持、URL 无法访问等）
3. 删除失败资源，重新上传

详情页顶部会显示红色的 **最近错误** 提示框。

## 编辑知识库

在知识库详情页，点击右上角的 **编辑** 按钮，修改名称、Slug 或描述，点击 **保存**。

修改 Slug 可能会影响使用该知识库的 Agent。

## 删除知识库

在知识库列表中，点击要删除的知识库行的 **删除** 按钮，确认后删除。

删除知识库会同时删除所有关联资源，且操作不可恢复。

## 绑定 Agent

知识库创建后，可以在 Agent 配置中绑定使用。具体操作请参考 [Agent 管理](../agents/)。
